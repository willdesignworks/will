border「框線」

css / November 26, 2023

border-width「寬度」、border-style「樣式」、border-color「色碼」、border-radius「圓角」、border-image「邊框-漸層」

Tags: border-style「樣式」 border-radius「圓角」 border-image「邊框-漸層」

．border 框線

border: 1px solid #444444; /*一行縮寫-寬度、樣式、 色碼*/

border-top: 5px solid #444444; /*上框線-寬度、樣式、 色碼*/

border-bottom: 1px solid #444444; /*下框線-寬度、樣式、 色碼*/

border-left: 2px dotted #444444; /*左框線-寬度、樣式、 色碼*/

border-right: 3px double #444444; /*左框線-寬度、樣式、 色碼*/
