React介紹, 常用軟體

React / January 05, 2024

React是 Facebook 所維護並開源的函式庫（或是有些人習慣用「前端框架」來統稱）
React 的開發依賴於 Node.js 和 npm（Node Package Manager）。

要使用React有幾種方法:
第一種使用:create-react-app
第二種使用:vite (前端框架建構工具)

第一種 create-react-app 使用步驟

  1. 終端機(Terminal)輸入npx create-react-app 我的專案名
  2. cd 你的專案名稱(進入到專案資料夾)
  3. npm install(安裝專案需要套件)
  4. npm start(啟動一個開發伺服器http://localhost:3000/)

第二種 vite 使用步驟

  1. 終端機(Terminal)輸入npm create vite@latest
    latest代表是最新版本,也能改成指定版本號
  2. Project name: 輸入專案名稱
  3. React(選擇需要安裝的前端框架)
  4. JavaScript(選擇語言,使用最標準的選項)
  5. 專案創建完稱, 資料夾會多檔案
  6. 依序輸入三道指令
  7. cd 你的專案名稱(進入到專案資料夾)
  8. npm install(安裝專案需要套件)
  9. npm run dev(啟動一個開發伺服器http://localhost:5173/)
  10. 瀏覽器貼上http://localhost:5173/有畫面就是成功

專案資料夾介紹

  1. node_modules (專案需要的套件)
  2. public (靜態資源,圖片,txt)
  3. src (存放專案原始碼)
  4. src -> assets (圖片檔)
  5. src -> App.jsx (JavaScriot XML 可以讓JS跟HTML結合再一起,組件放置處)
  6. src -> index.html (用來根伺服器動態修改html內容)
  7. src -> package.jeson (紀錄專案基本資訊,版本號,套件資料)

常用插件

  • React Router (根據不同的 URL，在畫面上渲染出相對應的 UI。)

    傳統的網站在進行換頁時（點擊連結 --> 網址改變 --> 更新頁面），瀏覽器會傳送一個 request
    給 server， 接著 server 會回傳一個 HTML 檔案給瀏覽器，這種換頁方式會需重新整理才能更新頁面，因此會產生空白頁，降低使用者體驗。

    而 SPA (Single Page Application）改善了傳統網頁換頁時的缺點，
    將頁面處理的工作全部都交給 client 端執行，因此換頁時不再需要傳送 request 給 server， 進而解決了空白頁的缺點，提升使用者體驗 。

    而使用 React 並搭配 React Router 就能輕鬆實作一個 SPA 網站。
    原本當 URL 改變時會向 server 發出請求，使用了 React Router 之後，React Router 則會開始接手管理 URL 發生改變時後續的工作， 不會傳送 request 給 server。

    在終端機輸入安裝指令

    npm install react-router-dom (npm install react-router-dom@5 安裝v5版本)

    基本介紹

    1. <BrowserRouter> 放在最外層
    2. <Routes> 用來包覆自己設定的 <Route>
    3. <Route> 的兩種屬性：
      • path 設定路徑，如 '/' 表示首頁，亦可用 index 取代
      • element 則指定要在該路徑顯示的畫面
    4. 使用 <Link> 來製作連結 ( <Link to='/網址home/'> HOME </Link> )

  • Prettier - Code formatter (整理程式碼-Format Document With...)

  • npm install react-bootstrap bootstrap

  • uuid npm (npm install uuid, 安裝完之後, 會產生唯一的編號)

    官網：https://www.npmjs.com/package/uuid
    import { v4 } from 'uuid'; (引入插件)

  • json-server-npm (模擬後端API)

    官網：https://www.npmjs.com/package/json-server
    npm install json-server
    npm run server

  • react-icons (終端機安裝)

    cd 專案名稱 (進入專案資料夾)
    npm install react-icons (安裝icons圖示)
    React Icons 網址: https://react-icons.github.io/react-icons/

ChatGpt安排學習

開始學習 React 是一個很好的選擇，因為它是一個非常流行且強大的 JavaScript 庫，
用於構建用戶界面。以下是一個逐步的學習課程安排，適合初學者：

  • 第一週：基礎知識和準備工作

    目標：了解基本概念並設置開發環境
    1. JavaScript 基礎
      • 熟悉基本語法（變數、陣列、對象、函數等）
      • 理解 ES6 特性（例如箭頭函數、解構賦值、模板字符串、let/const 等）
    2. HTML 和 CSS 基礎
      • HTML 基礎標籤和結構
      • CSS 選擇器、屬性和布局
    3. 設置開發環境
      • 安裝 Node.js 和 npm
      • 安裝一個代碼編輯器（例如 VSCode）
    4. 學習 React 的基本概念
      • 什麼是 React？
      • 了解組件、props 和 state

  • 第二週：React 基礎

    目標：構建基本的 React 應用
    1. 創建你的第一個 React 應用
      • 使用 Create React App 工具創建一個新的 React 專案
    2. React 組件
      • 函數組件 vs 類組件
      • state 的使用 (state 狀態-來記錄改變的狀態)
      • props 的使用 (props 傳遞數值-傳遞設定屬性或資料)
    3. JSX 語法
      • 理解 JSX 是什麼
      • 如何在 JSX 中嵌入表達式
    4. 事件處理
      • 添加事件處理器
      • 如何在 React 中處理事件

  • 第三週：進階主題

    目標：學習更高級的 React 概念和功能
    1. 條件渲染
      • if 判斷渲染
      • 使用三元運算符和邏輯運算符進行渲染
    2. 列表和 Key
      • 渲染列表
      • 理解 key 的重要性
    3. 表單處理
      • 控制組件和非控制組件
      • 處理表單提交
    4. 組件生命周期方法
      • 了解類組件中的生命周期方法（例如 componentDidMount、componentDidUpdate、componentWillUnmount）
      • 使用 hooks（例如 useEffect）處理函數組件中的副作用

  • 第四週：React hooks 和 Context API

    目標：掌握 hooks 和應用 Context API
    1. 基本 hooks
      • useState (讓 React 幫我們更新 state 改變的畫面)

        //引入
        import { usrState } from 'react'
        //列的解構賦值
        const [count(原本的內容,自訂一個名稱), setCount(設定改變的內容)] = useState(初始值);
      • useEffect (效果)
      • useRef ()
    2. 進階 hooks
      • useContext
      • useReducer
      • 自定義 hooks
    3. Context API
      • 創建和使用 Context
      • 提供者和消費者模式

  • 第五週：React 路由和狀態管理

    目標：學習如何在 React 中實現路由和管理全局狀態
    1. React Router
      • 安裝和設置 React Router
      • 配置路由和導航
      • 動態路由和路由參數
    2. 狀態管理
      • 使用 Context API 管理狀態
      • 使用 Redux 管理狀態
      • Redux 的基礎知識（store、actions、reducers）

  • 第六週：項目實戰和最佳實踐

    目標：應用所學知識構建一個完整的項目
    1. 項目實戰
      • 選擇一個小型項目（例如待辦事項應用、博客系統等）
      • 設計項目結構
      • 實現項目功能
    2. 最佳實踐
      • 組件設計和重用
      • 性能優化
      • 測試和調試

  • 第七週：部署和持續學習

    目標：學習如何部署 React 應用並保持學習更新
    1. 部署
      • 構建生產版本
      • 部署到 Netlify、Vercel 或其他平台
    2. 持續學習
      • 參與社區（例如 GitHub、Stack Overflow）
      • 閱讀 React 文檔和博客
      • 嘗試學習其他相關技術（例如 TypeScript、Next.js）
