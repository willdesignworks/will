HTML table表格

css / November 23, 2023

HTML table 標籤(tag) 用來建立表格，也就是用來呈現二維的資料表資訊。

width「寬度」 border「邊框」 bgcolor「背景色」 frame「外側邊框」 cellpadding「內距」 cellspacing「格子之間的距離」 align「對齊方式」 colspan「合併格子」

width= "300" /*控制 table 的寬度*/

這裡可以放表格內容 這裡可以放表格內容

 <tr>
  <td>這裡可以放表格內容</td>

 </tr>
</table>

frame= "hsides" /*顯示 上部 和 下部 的外側邊框*/

．table對齊方式

align= "right" /*水平對齊-靠右*/

valign= "top" /*垂直對齊-靠上*/

valign= "middle" /*垂直對齊-置中*/

valign= "bottom" /*垂直對齊-靠下*/
