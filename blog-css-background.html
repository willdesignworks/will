background「背景」

css / November 27, 2023

background-color「背景-顏色」、background-image「背景-圖片」、background-repeat「背景-重複」、background-position「背景-圖像起始位置」、background-attachment「背景-固定模式」、background-size「背景-大小属性」

Tags: background-color「背景色」 background-image「背景圖」 background-repeat「重複」 background-position「起始位置」 background-attachment「背景圖固定模式」 background-size「背景圖大小」 background-image「背景-漸層颜色」

．一行縮寫

background: url('網址or路徑') center center no-repeat #0011; /*背景圖路徑、起始位置x、起始位置y、不重複、背景色*/

．background-color「背景-顏色」

background-color: #0011; /*背景颜色*/

background-color: #0011;
<div style="background-color: #0011;"> ... </div>

．background-image「背景-圖片」

background-image: url('網址or路徑'); /*設定背景圖片、預設背景會重複*/

<style>
.background-image{
  background-image:url('網址or路徑');
}
</style>

<div class="background">background-image:url('網址or路徑')</div>

．background-repeat「背景-重複」

background-repeat: repeat; /*預設、在垂直方向和水平方向重複*/

background-repeat: repeat-x; /*在水平方向重複*/

background-repeat: repeat-y; /*在垂直方向重複*/

background-repeat: no-repeat; /*不重複、僅顯示一次*/

background-repeat: repeat;
background-repeat: repeat-x;
background-repeat: repeat-y;
background-repeat: no-repeat;

．background-attachment「背景-固定模式」

background-attachment: scroll; /*預設值，背景圖片會隨著滑鼠滾動而上下移動。*/

background-attachment: fixed; /*背景圖片固定不動，滑鼠上下滾動僅會移動網頁主體。記得要設定background-position*/

background-attachment:scroll;
background-attachment:fixed;

．background-size「背景-大小属性」

background-size: auto; /*預設值，背景圖-維持原本的大小*/

background-size: 50px 60px; /*背景圖-自訂大小*/

background-size: contain; /*背景圖-縮小至容器內呈現*/

background-size: cover; /*背景圖-放大至容器填滿*/

/*針對各種瀏覽器*/
-moz-background-size:cover; /*-moz-Firefox*/
-webkit-background-size:cover; /*-webkit-Opera,Chrome。*/
-o-background-size:cover; /*-o-IE*/

background-size:auto;
background-size:50px 60px;
background-size:contain;
background-size:cover;

．background-image 背景-漸層颜色

background: linear-gradient(90deg, #50d2d6 3%, #27a4b6 40%); /*線性漸層*/

background: radial-gradient(90deg, #50d2d6 3%, #27a4b6 40%); /*放射漸層*/

background: conic-gradient(90deg, #50d2d6 3%, #27a4b6 40%); /*圓錐形漸層 ( 新的 )*/

background: repeating-radial-gradient(90deg, #50d2d6 3%, #27a4b6 40%); /*放射重複漸層*/

background: repeating-linear-gradient(90deg, #50d2d6 3%, #27a4b6 40%); /*線性重複漸層*/

background: repeating-conic-gradient(90deg, #50d2d6 3%, #27a4b6 40%); /*圓錐形重複漸層 ( 新的 )*/

background: linear-gradient(方向, 顏色1 位置, 顏色2 位置); /*說明*/

background-image: linear-gradient(90deg, #50d2d6 3%, #27a4b6 40%); /*背景-漸層颜色*/
<style>
.background {
  background-image: linear-gradient(90deg, #50d2d6 3%, #27a4b6 40%);
}
</style>

<div class="background">background-image: linear-gradient(90deg, #50d2d6 3%, #27a4b6 40%); /*背景-漸層颜色*/</div>
