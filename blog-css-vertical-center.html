CSS 5種文字垂直居中

css / November 27, 2023

垂直置中一直是網頁設計師的痛，早期解決方法沒有那麼多，最常見到的就是height = line-height這種方式，可是這方式彈性很低，如果遇到多行文字就無法使用。然而隨著CSS的演進，越來越多種垂直置中的方法，這邊就介紹幾種實用性高的垂直置中方法。

Tags: 文字垂直置中「line-height」 文字垂直置中「display: table-cell 」 文字垂直置中「display:flex 」 文字垂直置中「transform: translateY(-50%) 」 文字垂直置中「top:calc 」

line-height 文字垂直置中

．display: table-cell 垂直置中方式

display: table; /*最外圍要一個div包起來*/

text-align: center; /*文字水平對齊*/

vertical-align: middle; /*利用支援儲存格垂直置中*/

table,margin失效

display: table; /**/

border-collapse: separate; /*邊框彼此間分開*/

border-spacing: 5px; /*邊框之間的距離*/

table,限制每隔寬度

display: table; /**/

table-layout: auto; /*自動寬度*/

table-layout: fixed; /*全部等寬*/

使用表格或假裝表格
css display: table-cell 垂直置中方式

display: table-cell 垂直置中方式-

<style>
.table-cell {
  display: table;
  text-align: center;
}
.table-cell .in {
  width: 300px;
  height:150px;
  display: table-cell;
}
</style>

<div class="table-cell">
 <span class="in">
 使用表格或假裝表格

 </span>

．translateY 垂直置中方式

position: relative; /**/

top: 50%; /**/

transform: translateY(-50%); /**/

css3 translateY多行文字的垂直置中技巧Top: 50% 的招式，讓你的物件上方能產生固定％數的距離由於 translate的百分比單位是利用物件自身的尺寸作為 100% ，這樣讓我們要利用物件自身寬高

display: table-cell 垂直置中方式

<style>
.table-cell {
  width: 300px;
  height:300px;
  position: relative;
}
.table-cell .in {
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
}
</style>

．css3 calc 垂直置中方式

position: relative; /**/

top: calc((100% - 150px) / 2); /**/

css3 calc多行文字的垂直置中技巧利用 calc() 這個「計算機」方法，來將百分比即時且動態的計算出實際要的什麼高度但這個方法需要注意的是太過大量的使用的話，網頁效能會是比較差

css3 calc 垂直置中方式

<style>
.calc {
  width: 300px;
  height:300px;
  position: relative;
  top: calc((100% - 150px) / 2);
}
</style>

</div>
