偽類-錨點、狀態、child、of-typ

css / November 28, 2023

:link「錨點-未訪問的連結」、:visited 「錨點-已訪問的連結」、:hover「錨點-滑鼠其上的元素」、:active「錨點-活動的連結」、:focus「獲取焦點的輸入欄位」、:enabled「狀態-啟用的表單」、:disabled「狀態-禁用的表單」、:first-child「子元素-第一個」、:first-child「子元素-最後一個」、:nth-child(數字)「子元素-第(_)個」、:nth-child(2n)「子元素-偶數」、:nth-child(2n+1)「子元素-奇數」、:nth-last-child(數字)「子元素-後面數來」、:only-child「子元素-只有一個」、:first-of-type「同元素-第一個」、:last-of-type「同元素-最後一個」、:nth-of-type()「同元素-第(_)幾個」、:nth-last-of-type()「同元素-後面數過來第(_)個」、:only-of-type「同元素-只有這種元素」

Tags: 偽類-錨點 偽類-狀態 偽類-child 偽類-of-type

．偽類 (錨點)

css2-偽類 (錨點)

:link /*選擇未訪問的連結*/

:visited /*選擇已訪問的連結*/

:hover /*滑鼠其上的元素*/

:active /*選擇活動的連結*/

:focus /*選擇獲取焦點的輸入欄位*/

:link 未訪問的連結
<style>
.a_link a:link{
  color: #FF5F00;
}
</style>

<div class=".a_link">
 <a href="#">:link未訪問的連結</a>
</div>
:visited 已訪問的連結,請點一下
<style>
.a_visited a:visited{
  color: #FF5F00;
}
</style>

<div class=".a_visited">
 <a href="#">:visited已訪問的連結,請點一下</a>
</div>
:hover滑入變色
<style>
.a_hover a:hover{
  color: #FF5F00;
}
</style>

<div class=".a_hover">
 <a href="#">:hover滑入變色</a>
</div>
:active 選擇活動的連結,按住別放
<style>
.a_active a:active{
  color: #FF5F00;
  background: #FFED00;
}
</style>

<div class=".a_active">
 <a href="#">:active選擇活動的連結,按住別放</a>
</div>
input:focus選擇獲取焦點的輸入欄位 
<style>
.focus input:focus{
  background: #FFED00;
}
</style>

<div class=".focus">
 input:focus<input type="text" name="UserName">
</div>

．偽類child (結構選擇器-子元素)

css3-偽類child (結構選擇器-子元素)

:first-child /*「第一個」子元素,:前面有一個空白*/

:last-child /*「最後一個」子元素,:前面有一個空白*/

:nth-child(數字) /*「第(_)個」子元素,:前面有一個空白*/

:nth-child(2n) /*挑出「偶數」的子元素(2的倍數),:前面有一個空白*/

:nth-child(2n+1) /*挑出「奇數」的子元素,:前面有一個空白*/

:nth-last-child(數字) /*「後面數來」第幾個子元素,:前面有一個空白*/

:only-child /*「只有一個」子元素,:前面有一個空白*/

．偽類of-type (結構選擇器-同元素)

css3-偽類of-type (結構選擇器-同元素)

:first-of-type /*同元素-第一個,:前面有一個空白*/

:last-of-type /*同元素-最後一個,:前面有一個空白*/

:nth-of-type() /*同元素-的第()幾個,:前面有一個空白*/

:nth-last-of-type() /*同元素-後面數過來第()幾個,:前面有一個空白*/

:only-of-type /*只有這種元素,:前面有一個空白*/
